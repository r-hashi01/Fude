# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc] as its
code of conduct. Participation in the `fude` community — issues, pull
requests, discussions, and any other project spaces — is governed by
that document.

The full text is available at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

If you experience or witness behaviour that violates the Contributor
Covenant, please report it to the project maintainer at
<r.hashimoto@dify.ai>. Reports are handled privately and will not be
shared beyond the maintainer without the reporter's agreement.

Enforcement follows the "Enforcement Guidelines" section of the
Contributor Covenant 2.1 linked above.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
